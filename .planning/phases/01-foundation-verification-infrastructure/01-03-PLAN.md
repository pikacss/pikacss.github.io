Create `scripts/run-all-checks.sh` that orchestrates all validation checks:

1. Run ESLint markdown validation: `pnpm lint`
2. Run link checker: `bash scripts/check-links.sh`
3. Run file reference validator: `bash scripts/check-file-refs.sh`
4. Run placeholder detector: `bash scripts/check-placeholders.sh`

For each check:
- Print section header (e.g., "=== Running Link Validation ===")
- Run check and capture output
- Report pass/fail status
- Collect all outputs for final summary

Exit behavior:
- If all checks pass: exit 0
- If any check fails: print summary of failures, exit 1

```
=== Documentation Validation ===

[1/4] ESLint markdown validation...
✓ Passed

[2/4] Internal link validation...
✗ Failed: 3 broken links
docs/guide/basics.md:15: Broken link to './nonexistent.md'

[3/4] File reference validation...
✓ Passed

[4/4] Placeholder detection...
✗ Failed: 12 placeholders found

=== Summary ===
✓ 2 passed
✗ 2 failed
```

Create `.github/workflows/docs-validation.yml` GitHub Actions workflow:

Trigger on:
- push to main branch
- pull_request to main branch
- paths: `docs/**/*.md`, `**/README.md`, `skills/**/*.md`, `**/AGENTS.md`, `scripts/check-*.sh`

Jobs:
1. **structural-validation**
   - Checkout repository
   - Setup Node.js (use version from package.json engines or latest LTS)
   - Install dependencies: `pnpm install`
   - Run unified validation: `bash scripts/run-all-checks.sh`
   - Upload validation report as artifact

IMPORTANT:
- Use existing pnpm setup patterns from any existing workflows in .github/workflows/
- Don't check external links (rate limits, unreliable)
- Only internal structural validation
- Should complete in <2 minutes
- Block PR merge if validation fails

If no existing workflows, use standard GitHub Actions patterns:
```yaml
name: Documentation Validation
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]
    paths: [docs/**, "**.md", scripts/check-*.sh]
jobs:
  validate:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: pnpm/action-setup@v2
      - uses: actions/setup-node@v4
        with:
          node-version: 20
          cache: pnpm
      - run: pnpm install
      - run: bash scripts/run-all-checks.sh
```
  </action>

5. Confirm workflow appears in GitHub Actions tab
