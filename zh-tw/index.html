PikaCSSAtomic CSS-in-JS 引擎

用 JS 物件撰寫樣式，產生零執行階段成本的 atomic CSS。

開始使用

⚡

零執行階段成本

所有 CSS 都在建置時期產生，正式環境沒有任何執行階段負擔。

✍️

CSS-in-JS 語法

用熟悉的 CSS 屬性名稱撰寫樣式，不必背誦 utility class 名稱。

🧬

Atomic CSS 輸出

每一條 CSS 宣告都會變成一個原子 class，讓重複使用最大化、bundle 體積最小化。

🔧

完全可自訂

選擇器、shortcut、變數、關鍵影格，以及強大的外掛系統。

🌐

不綁定框架

支援 Vite、Webpack、Rspack、esbuild、Rollup、Rolldown 與 Nuxt。

🤖

TypeScript 優先

為每個 CSS 屬性與自訂設定提供完整的自動完成與型別檢查。

直接看程式碼 ​

用單純的 CSS 屬性名稱撰寫樣式，在建置時期得到 atomic CSS：

ts
const className = pika({
	color: 'red',
	fontSize: '16px',
})
css
@layer utilities {
  .pk-a {
    color: red;
  }
  .pk-b {
    font-size: 16px;
  }
}
