Zero Runtime

All CSS is generated at build time. No runtime overhead in production.

✍️

CSS-in-JS Syntax

Write styles using familiar CSS property names. No utility class names to memorize.

🧬

Atomic CSS Output

Each CSS declaration becomes an atomic class, maximizing reuse and minimizing bundle size.

🔧

Fully Customizable

Selectors, shortcuts, variables, keyframes, and a powerful plugin system.

🌐

Framework Agnostic

Works with Vite, Webpack, Rspack, esbuild, Rollup, Rolldown, and Nuxt.

🤖

TypeScript First

Full autocomplete and type checking for every CSS property and custom config.

Show Me the Code ​

Write styles with plain CSS property names, get atomic CSS at build time:

ts
const className = pika({
	color: 'red',
	fontSize: '16px',
})
css
@layer utilities {
  .pk-a {
    color: red;
  }
  .pk-b {
    font-size: 16px;
  }
}
